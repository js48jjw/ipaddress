웹사이트 정보

이 웹사이트는 사용자의 공인 IP 주소와 현재 위치 정보를 확인할 수 있는 도구입니다.

기능 설명

  • 공인 IP 주소: 인터넷에서 사용되는 고유한 주소로, 외부에서 사용자를 식별하는 데 사용됩니다.
  • 현재 위치: IP 주소를 기반으로 사용자의 대략적인 위치 정보를 표시합니다.
  • 새로고침 버튼: IP 주소 및 위치 정보를 다시 가져옵니다.

기술 정보

  • 공인 IP 주소는 ipify.org API를 통해 가져옵니다.
  • 현재 위치 정보는 ipapi.co 서비스를 사용하여 IP 주소로부터 위치 정보를 가져옵니다.
  • 배경 이미지는 Picsum Photos에서 제공되며, 매일 오전 6시에 변경됩니다.

사용 방법

  1. 홈 화면에서 현재 IP 주소와 위치 정보를 확인할 수 있습니다.
  2. IP 주소나 위치 정보가 변경되었을 경우 "새로고침" 버튼을 클릭하여 최신 정보를 가져올 수 있습니다.
  3. "정보" 탭에서 웹사이트의 기능과 사용 방법을 확인할 수 있습니다.
